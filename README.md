# Password Tester

## Overview
Password Tester is a lightweight web application that helps users assess the strength of their passwords in real-time. It checks for common passwords, evaluates length, character variety, and predictable patterns, and provides both a visual strength bar and a textual status to guide users toward creating safer passwords.

## Features
- Real-time password strength evaluation.
- Detects common and weak passwords.
- Scores passwords based on length, character diversity, and patterns.
- Provides a visual bar and status message for instant feedback.

## Installation and Use
1. Clone or download the repository.
2. Navigate to the project folder in your terminal.
3. Install required packages:

```bash
pip install flask flask-cors
```
4. Start the application:

```bash
python app.py
```
5. Open your browser and go to to use it
